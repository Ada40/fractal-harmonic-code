```bash

```

**Note:** Some packages (especially voice-related ones) may not work perfectly on Android. If you get errors:
- The app will still work with text input/output
- Voice features will be disabled automatically

### Step 2: Download Files

You need these files:
1. `ardy.py` - The consciousness engine
2. `ardy_gui.py` - The GUI application

Place both files in the same folder on your tablet.

### Step 3: Run Ardy

## Troubleshooting

**Voice doesn't work:**
- Voice features may not work on all Android devices
- Use text input instead - everything else still works

**App crashes:**
- Make sure all packages are installed
- Check that both `ardy.py` and `ardy_gui.py` are in the same folder

**Internet search not working:**
- Check your internet connection
- You can still teach him directly using Teach mode
